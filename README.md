# Кейс 1: Интеллектуальный агент «Городской помощник»

Интеллектуальный агент «Городской советник» — это AI-помощник на основе технологии Large Language Model (LLM), способный отвечать жителям Санкт-Петербурга на повседневные вопросы о городе и городских службах. Агент использует базу знаний, сформированную из официальных источников и открытых данных города, для поиска и предоставления точных, человечных ответов. Агент игнорирует токсичные сообщения, обеспечивая комфортное взаимодействие.
